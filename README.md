# Feature Engineering for Machine Learning 🚀

This repository contains practical implementations of essential Feature Engineering techniques used in Machine Learning.

## 📌 Topics Covered

- Handling Missing Values
- Handling Imbalanced Dataset (Upsampling & Downsampling)
- SMOTE (Synthetic Minority Oversampling Technique)
- Handling Outliers using Python
- Encoding Techniques:
  - One Hot Encoding (OHE)
  - Label Encoding
  - Ordinal Encoding
  - Target Guided Ordinal Encoding

## 🛠️ Tools & Libraries

- Python
- Pandas
- NumPy
- Scikit-learn
- Imbalanced-learn (SMOTE)

## 🎯 Goal

To build a strong foundation in feature engineering by applying real-world preprocessing techniques before training machine learning models.

## 📂 Use Cases

- Data Preprocessing
- Improving Model Accuracy
- Handling Real-world Dataset Challenges
